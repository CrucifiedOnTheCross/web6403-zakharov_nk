# Лабораторная работа №3

Цель: изучение основ JavaScript через реализацию набора функций и проверку их корректности тестами.

## Структура

- Основной код и тесты находятся в `../js-tasks`.
  - Реализация: `../js-tasks/src/core.js`
  - Тесты: `../js-tasks/src/core.spec.js`

## Как запустить

1. Перейдите в папку `../js-tasks`.
2. Установите зависимости: `npm install`.
3. Запустите тесты: `npm test`.

Все текущие тесты проходят успешно.
