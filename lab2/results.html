Комплексная оценка производительности моделей

Для объективной оценки качества разработанных моделей классификации был использован набор стандартных и специфичных для медицинской диагностики метрик. В данном разделе представлены определения ключевых показателей, результаты сравнительного анализа моделей и детальный разбор ошибок классификации с помощью матрицы ошибок.

Ключевые метрики оценки

Accuracy (Точность)

Доля всех правильно классифицированных объектов. Является базовой метрикой, однако её интерпретация может быть затруднена в условиях несбалансированных классов.

Формула: (TP + TN) / (TP + TN + FP + FN)

Precision (Точность предсказания)

Доля истинно положительных объектов среди всех, которые модель классифицировала как положительные. Высокая точность важна для минимизации ложных диагнозов.

Формула: TP / (TP + FP)

Recall (Чувствительность, Полнота)

Доля истинно положительных объектов, которые были успешно обнаружены моделью. Является критически важной метрикой в медицине для минимизации пропуска опасных заболеваний.

Формула: TP / (TP + FN)

Specificity (Специфичность)

Доля истинно отрицательных объектов, которые были правильно идентифицированы. Показывает способность модели корректно определять отсутствие заболевания.

Формула: TN / (TN + FP)

F1-Score (F-мера)

Гармоническое среднее между Precision и Recall. Позволяет получить сбалансированную оценку, особенно полезную при работе с несбалансированными наборами данных.

Формула: 2 * (Precision * Recall) / (Precision + Recall)

ROC-AUC

Площадь под кривой ошибок (Receiver Operating Characteristic). Демонстрирует качество модели независимо от выбранного порога классификации. Значение, близкое к 1.0, указывает на высокую разделяющую способность.

Агрегированные метрики для мультиклассовой классификации

Поскольку задача является мультиклассовой, для получения единой оценки по всем классам используются стратегии усреднения:

  • Macro Average: Рассчитывает метрику для каждого класса отдельно, а затем находит их среднее арифметическое. Все классы считаются равнозначными, независимо от их размера.
  • Weighted Average: Также рассчитывает метрику для каждого класса, но усредняет их с учётом веса каждого класса (т.е. количества экземпляров). Это более релевантно для несбалансированных данных.
