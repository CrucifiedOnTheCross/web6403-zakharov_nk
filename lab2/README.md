# Лабораторная работа №2 — HTML/CSS

Простой сайт про классификацию дерматоскопических изображений (HAM10000). На страницах коротко описаны данные, модели, расширение данных и результаты.

Где лежит
- Папка: `lab2/`
- Страницы: `index.html`, `dataset.html`, `model.html`, `results.html`
- Стили: `styles.css`
- Картинки: `img/`

Что на страницах
- Главная: верхний блок без картинки, список задач, 3 карточки с кнопками перехода.
- Датасет: краткое описание HAM10000 и таблица.
- Модель: простое описание подхода и расширения данных; есть картинка `stylegan2-ada-teaser-1024x252.png`.
- Результаты: метрики и краткое сравнение.

Особенности верстки
- Тёмная тема, понятные отступы.
- Кнопки для перехода между разделами.
- Картинки в карточках занимают всю ширину блока.
- Нижний отступ для основного контента: `main.container { padding-bottom: 80px; }
