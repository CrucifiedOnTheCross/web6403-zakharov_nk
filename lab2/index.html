Применение глубоких свёрточных нейронных сетей для классификации новообразований кожи

Ранняя и точная диагностика злокачественных новообразований кожи, таких как меланома, является ключевым фактором успешного лечения. Современные методы глубокого обучения, в частности свёрточные нейронные сети (CNN), демонстрируют высокий потенциал в качестве систем поддержки принятия врачебных решений, способных анализировать дерматоскопические изображения с высокой точностью.
Данный проект посвящён разработке и оценке ансамбля моделей на основе передовых архитектур CNN для автоматической классификации семи различных типов поражений кожи. В качестве основы используется публичный датасет HAM10000.

Цели и задачи проекта

  • Исследование и подготовка данных: Анализ датасета HAM10000, его предобработка и стратифицированное разделение на выборки.
  • Обучение базовых моделей: Применение техники трансферного обучения для тонкой настройки архитектур (ResNet50, DenseNet201, EfficientNetB0).
  • Борьба с дисбалансом классов: Использование аугментации и генерации синтетических данных с помощью StyleGAN2-ADA для повышения робастности моделей.
  • Построение ансамбля: Разработка и оценка ансамблевых методов (Soft Voting, Stacking) для повышения итоговой точности классификации.
  • Анализ результатов: Комплексная оценка производительности с помощью релевантных метрик и детальный разбор ошибок.
