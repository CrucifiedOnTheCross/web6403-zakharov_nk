Описание и структура набора данных

В основе данного исследования лежит общедоступный набор данных HAM10000 ("Human Against Machine with 10000 Training Images"). Этот датасет представляет собой коллекцию дерматоскопических изображений распространённых пигментных поражений кожи и является признанным стандартом для задач автоматической классификации в дерматологии.

Описание классов

Датасет включает 7 классов дерматологических поражений, каждый из которых представляет собой отдельную диагностическую категорию:

  • akiec: Актинические кератозы и внутриэпителиальная карцинома — предраковые поражения, вызванные длительным воздействием солнца.
  • bcc: Базально-клеточная карцинома — наиболее распространённый вид рака кожи.
  • bkl: Доброкачественные кератозоподобные поражения (себорейный кератоз, лихеноидный кератоз).
  • df: Дерматофиброма — доброкачественное новообразование кожи.
  • mel: Меланома — наиболее опасный вид рака кожи.
  • nv: Меланоцитарные невусы (родинки) — доброкачественные скопления меланоцитов.
  • vasc: Сосудистые поражения (ангиомы, ангиокератомы, пиогенные гранулемы).

Предобработка и разделение данных

Перед подачей в нейронную сеть все изображения прошли через конвейер предобработки, который был адаптирован под требования каждой конкретной архитектуры. Этот процесс включал:

  1. Изменение размера: Изображения приводились к разрешению, ожидаемому на входе каждой предобученной модели (например, 224x224 для ResNet/DenseNet, или специфичные размеры для EfficientNet).
  2. Нормализация: Значения пикселей нормализовались с использованием средних значений и стандартных отклонений, на которых обучались исходные модели (как правило, это статистики набора данных ImageNet). Этот шаг является обязательным для корректной работы моделей в режиме трансферного обучения.

Для обеспечения объективной оценки модели, исходный набор данных был разделён на обучающую (training), валидационную (validation) и тестовую (test) выборки. Разделение проводилось с использованием стратификации для сохранения исходного распределения классов во всех трёх подвыборках.
